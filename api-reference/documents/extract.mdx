<RequestExample>




```python python
from retab.client import Retab

reclient = Retab()
response = reclient.documents.extract(
    json_schema = "freight/booking_confirmation_json_schema.json",
    document = "freight/booking_confirmation.jpg",
    model="gpt-4.1-nano",
    temperature=0
)
```


```bash curl
curl -X 'POST' \
  'https://api.retab.dev/v1/documents/extract' \
  -H "API-Key: $RETAB_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
  "model": "string",
  "json_schema": {},
  "stream": false,
  "seed": null,
  "temperature": 0,
  "modality": "native",
  "document": {
    "id": "string",
    "name": "string",
    "mime_type": "string",
    "content": "string"
  }
}'
```
