<RequestExample>

```python python
from uiform import UiForm
from uiform.types.mime import MIMEData

client = UiForm()


# 2) Ask UiForm to infer a JSON Schema from that document.
response = client.schemas.generate(
    documents=["sample_invoice.pdf"],
    model="gpt-4o-mini",      # or any model your plan supports
    temperature=0.0,          # keep the generation deterministic
    modality="native",        # "native" = let the API decide best modality
)

print("Generated JSON Schema:")
print(response.json_schema)
```
