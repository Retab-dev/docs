<RequestExample>


```python python
from uiform import UiForm, MIMEData

uiclient = UiForm()

# Load document as MIMEData
with open("invoice.pdf", "rb") as f:
    document = MIMEData.from_bytes(f.read(), filename="invoice.pdf")

completion = uiclient.processors.submit(
    id="prc_01G34H8J2K",
    document=document
)

# Access extracted data
extracted_data = completion.choices[0].message.parsed
print(extracted_data)
```


```bash curl
curl -X 'POST' \
  'https://api.uiform.com/v1/processors/{processor_id}/submit' \
  -H 'accept: application/json' \
  -H 'Api-Key: <your-api-key>' \
  -F 'document=@invoice.pdf' \
  -F 'stream=false'
```
