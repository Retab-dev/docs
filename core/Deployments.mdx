
In this section, we will see how to use the methods of the `client.deployments` module. Please check the [API Reference](https://docs.uiform.com/api-reference/deployments/links/create) for more details.

---
## Introduction

A deployment allows you to analyze files and send the extracted data to your backend. You can create 4 types of deployments:

- **Links**: Share links to upload files.
- **Mailboxes**: Set up mailboxes to forward emails to.
- **Endpoints**: Create an endpoint to send data to.
- **Outlook Plugin**: Create and install an Outlook plugin to extract data from emails.

You can create, monitor, update, delete and test all the deployments in the [dashboard](https://www.uiform.com/dashboard/deployments).

## Link
----
### Object

<ResponseField name="Object" type="Link">
An Link object containing the extraction link configuration.
  <Expandable title="properties">
    <ResponseField name="object" type="string">
      The type of object. Always "deployment.link".
    </ResponseField>

    <ResponseField name="id" type="string">
      Unique identifier for the deployment link, prefixed with "lnk_".
    </ResponseField>

    <ResponseField name="name" type="string">
      Name of the link.
    </ResponseField>

    <ResponseField name="password" type="string">
      Optional password to access the link.
    </ResponseField>

    <ResponseField name="webhook_url" type="HttpUrl">
      URL of the webhook to send the data to.
    </ResponseField>

    <ResponseField name="webhook_headers" type="Dict[str, str]">
      Headers to send with the webhook request.
    </ResponseField>

    <ResponseField name="modality" type="Modality">
      Processing modality for the documents.
    </ResponseField>

    <ResponseField name="image_settings" type="ImageSettings">
      Preprocessing operations applied to images before sending to the LLM.
    </ResponseField>

    <ResponseField name="model" type="string">
      Model used for chat completion.
    </ResponseField>

    <ResponseField name="json_schema" type="Dict[str, Any]">
      JSON schema format used to validate the output data.
    </ResponseField>

    <ResponseField name="temperature" type="float">
      Temperature for sampling. Defaults to 0.0 if not provided.
    </ResponseField>

    <ResponseField name="updated_at" type="datetime">
      UTC timestamp of when the config was last updated.
    </ResponseField>
  </Expandable>
</ResponseField>
