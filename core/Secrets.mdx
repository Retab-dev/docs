

You can store your API keys securely in the [UiForm Dashboard](https://uiform.com/dashboard/settings) or using the SDK. All secrets are encrypted in transit using TLS 1.2+ and encrypted at rest using AES-256 encryption. Our security measures ensure your sensitive data remains protected throughout its lifecycle.

---




## External API Keys


To use the full capabilities of UiForm, you need to store securely your AI provider API keys in the [UiForm Dashboard](https://uiform.com/dashboard/settings) or using the SDK.



### Object
<ResponseField name="Object" type="ExternalAPIKey">
A model containing API key information for external AI providers.
  <Expandable title="properties">
    <ResponseField name="provider" type="OpenAI | Anthropic | Gemini | xAI">
      The AI provider this key is for.
    </ResponseField>

    <ResponseField name="is_configured" type="bool">
      Whether the API key is properly configured.
    </ResponseField>

    <ResponseField name="last_updated" type="datetime">
      Optional UTC timestamp of when the key was last updated.
    </ResponseField>
  </Expandable>
</ResponseField>

### Create

<CodeGroup>
```python Request
from uiform import UiForm

uiclient = UiForm()
api_key = uiclient.secrets.external_api_keys.create(
    provider="OpenAI",
    api_key="sk-..."
)
```
</CodeGroup>

### Get

<CodeGroup>
```python Request
from uiform import UiForm

uiclient = UiForm()
api_key = uiclient.secrets.external_api_keys.get(
    provider="OpenAI",
)
```
</CodeGroup>

### List

<CodeGroup>
```python Request
from uiform import UiForm

uiclient = UiForm()
list_api_keys = uiclient.secrets.external_api_keys.list()
```
</CodeGroup>

### Delete

<CodeGroup>
```python Request
from uiform import UiForm

uiclient = UiForm()
uiclient.secrets.external_api_keys.delete(
    provider="OpenAI",
)
```
</CodeGroup>
