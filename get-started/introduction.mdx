UiForm is a **modern**, **flexible**, and **AI-powered** open-source form generation library that helps you:

- Create dynamic forms from JSON schemas with zero boilerplate
- Style forms easily with Tailwind CSS
- Add AI capabilities for automated form filling and data extraction
- Customize layouts and validation rules
- Monitor form performance and user interactions

## Quickstart


<Steps>
  <Step title="Generate JSON Schema">
    Use **Pydantic** to generate a JSON Schema for your backend data structure.
  </Step>
  <Step title="Integrate with React">
    Integrate the JSON Schema with the **UiForm** component in React to render a fully functional form.
  </Step>
</Steps>





### Generate a JSON Schema

UiForm is compatible with any JSON Schema following the OpenAPI specification. For example, we can create a JSON Schema using **pydantic** :


<CodeGroup>
```python JSON Schema generation
from pydantic import BaseModel
import json

class User(BaseModel):
    first_name: str
    last_name: str
    email: str
    age: int
    address: str

with open("jsonSchema.json", "w") as f:
    f.write(json.dumps(User.model_json_schema()))
```

```json schema.json
{
    "properties": {
        "first_name": {
            "title": "First Name",
            "type": "string"
        },
        "last_name": {
            "title": "Last Name",
            "type": "string"
        },
        "email": {
            "title": "Email",
            "type": "string"
        },
        "age": {
            "title": "Age",
            "type": "integer"
        },
        "address": {
            "title": "Address",
            "type": "string"
        }
    },
    "required": [
        "first_name",
        "last_name",
        "email",
        "age",
        "address"
    ],
    "title": "User",
    "type": "object"
}
```
</CodeGroup>

The schema is saved as `jsonSchema.json`, ready to be imported into your React project.


### Render a Form with UiForm


To get started, install the `@UiForm` package using npm:

```bash
npm install @UiForm
```

Then, import the `UiForm` component and pass the schema as a prop.

```jsx App.jsx
import { UIForm } from 'uiform'
import { JSONSchema7 } from "json-schema";
import schema from './schema.json'
import "uiform/dist/index.css";

function App() {
    return (
        <div className='bg-slate-100 p-4'>
            <UIForm
                schema={schema as JSONSchema7}
                onSubmit={(data) => console.log("data", data)}
                onChange={(data) => console.log("data", data)}
                className='text-[#0A2540] shadow-lg bg-white border-[#0A2540] border rounded-sm'
            />
        </div>
    );
}

export default App;
```


- **`schema`**: The JSON Schema imported from the file.
- **`onSubmit`**: Callback that provides form data when submitted.
- **`className`**: Allows you to customize the form's appearance using Tailwind CSS.


And that's it ! Your form is generated based on the JSON Schema. Each field corresponds to a property defined in the Pydantic model, complete with client-side validation.
