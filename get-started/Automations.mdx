## Verify Event
---
If you want to verify the event you can use the `verify_event` method. The `WEBHOOKS_SECRET` can be generated in the [UiForm Dashboard](https://uiform.com/dashboard/settings).

<CodeGroup>
```python FastAPI
import os

from fastapi import FastAPI, Request, Response, HTTPException
from uiform import UiForm

uiclient = UiForm()
app = FastAPI()

@app.post("/webhook")
async def webhook_handler(request: Request):
    try:
        # Read payload and signature from request
        payload = await request.body()
        signature_header = request.headers.get("UiForm-Signature")

        if not signature_header:
            raise HTTPException(status_code=400, detail="Missing UiForm-Signature header")

        # Verify the signature and process the event
        uiclient.automations.verify_event(
            event_body=payload,
            event_signature=signature_header,
            secret=os.getenv("WEBHOOKS_SECRET"),
        )

        return Response(status_code=200)
    except Exception as e:
        # Handle errors (optional: log the error)
        return Response(status_code=400, content=f"Webhook error: {str(e)}")
```

## Extraction Link

### Object

<ResponseField name="Object" type="Link">
An Link object containing the extraction link configuration.
  <Expandable title="properties">
    <ResponseField name="object" type="string">
      The type of object. Always "extraction_link".
    </ResponseField>

    <ResponseField name="id" type="string">
      Unique identifier for the extraction link, prefixed with "el_".
    </ResponseField>

    <ResponseField name="name" type="string">
      Name of the link.
    </ResponseField>

    <ResponseField name="password" type="string">
      Optional password to access the link.
    </ResponseField>

    <ResponseField name="webhook_url" type="HttpUrl">
      URL of the webhook to send the data to.
    </ResponseField>

    <ResponseField name="webhook_headers" type="Dict[str, str]">
      Headers to send with the webhook request.
    </ResponseField>

    <ResponseField name="modality" type="Modality">
      Processing modality for the documents.
    </ResponseField>

    <ResponseField name="image_settings" type="ImageSettings">
      Preprocessing operations applied to images before sending to the LLM.
    </ResponseField>

    <ResponseField name="model" type="string">
      Model used for chat completion.
    </ResponseField>

    <ResponseField name="json_schema" type="Dict[str, Any]">
      JSON schema format used to validate the output data.
    </ResponseField>

    <ResponseField name="temperature" type="float">
      Temperature for sampling. Defaults to 0.0 if not provided.
    </ResponseField>

    <ResponseField name="updated_at" type="datetime">
      UTC timestamp of when the config was last updated.
    </ResponseField>
  </Expandable>
</ResponseField>
