# AI-Powered Forms Documentation

## Introduction

**For Paying Users Only**

To get the most out of this tutorial, ensure you have the following:

1. A functioning **UiForm**.
2. A **UiForm API key**.
3. An API key from the AI provider of your choice (OpenAI, Claude, Grok, Mistral, etc.).
4. A properly defined `.env` file with your **UiForm API key**.

### Example `.env` File

```env
UiForm_API_KEY=your_UiForm_api_key_here
OPENAI_API_KEY=your_ai_provider_api_key_here
```

The `.env` file securely stores sensitive keys and URLs required for AI-powered features. Replace the placeholder values with your actual keys.

## FormProps Interface

The following `FormProps` interface defines the key properties for the UiForms

```typescript
interface FormProps {
    schema: Record<string, any>;
    schemaWithRequired: Ajv.JSONSchemaType<any>;
    layout?: Layout;
    initialData?: Record<string, any>;
    is_file_upload_enabled: boolean;
    initialDataIsStreaming: boolean;
    uncertainties?: Record<string, any>;
    appearance: setup;
    onChange: (formValues: Record<string, any>, errors: FieldErrors<Record<string, any>>) => void;
    onSubmit: SubmitHandler<Record<string, any>>;
    language?: string;
    translations?: Record<string, any>;
}
```

Each property ensures flexibility and robustness when configuring the form.

- \`\`: Defines the form structure via JSON Schema.
- \`\`: Pre-populates fields with default values.
- \`\`: Handles form submission.
- \`\`: Allows for visualizing AI uncertainty in field values.
